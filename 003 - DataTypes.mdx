# Data Types 

As clear from the name, data types are type of data and a [variable](./002%20-%20Variables.mdx) must be a specified data type.

## Primitive Data Types

Primitive data types are simple and easy to work with

There are 8 types of primitive data types 

### 1 byte
`byte` can store numbers from  -128 to 127 (1 byte)

#### Syntax example using `byte`

```java
public class Main {
  public static void main(String args[]) {

    byte Num = 100;
    
    System.out.println(Num);
  }
}
```
> Output of the above code will be 100


### 2. short
`short` can store number from -32768 to 32767 (2 bytes)

#### Syntax example using 

```java
public class Main {
  public static void main(String args[]) {

    short Num = 6000;
    
    System.out.println(Num);
  }
}

```
> Output of the above code will be 6000

### 3. int
`int` can store numbers from -2147483648 to 2147483647 (4 bytes)

#### Syntax example using int

```java
public class Main {
  public static void main(String args[]) {

    int Num = 111000;

    System.out.println(Num);
  }
}
```
> Output of the above code will be 111000

### 4. long
`long` can store numbers from -9223372036854775808 to 9223372036854775807 (8 bytes), `long` usually used when the int value is large 

> [!IMPORTANT] 
> Long data type values must be end with "L"

#### Syntax example using long

```java
public class Main {
  public static void main(String args[]) {

    long Num = 10000000000L;
    
    System.out.println(Num);
  }
}

```
> Output of the above code will be 10000000000

### 5. double
`double` can store decimal number values, upto 16 decimals (8 bytes)

#### Syntax example using double

```java
public class Main {
  public static void main(String args[]) {

    double Num = 16.91;
    
    System.out.println(Num);
  }
}

```
> output of the above code will be 16.91

this is for situation in which he have to choose one from two options like yes/no, enable/disable, on/off
